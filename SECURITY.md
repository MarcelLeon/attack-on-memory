# Security Policy

## Supported versions

Currently supported: `0.1.x` (main branch).

## Reporting a vulnerability

Please do **not** open public issues for sensitive reports.

Contact maintainers privately with:
- Impact summary
- Reproduction steps
- Proof-of-concept (if possible)
- Suggested remediation

We will acknowledge within 72 hours and provide remediation timelines.

## Security baseline for this repo

- Branch protection on `main`
- Required PR reviews
- CI required before merge
- Dependabot enabled
- OSSF Scorecard workflow enabled
